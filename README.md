## 🌟 Featured Project

### 🤖 AI Interview Preparation & Evaluation System
*An intelligent, voice-enabled SaaS platform designed to help students prepare for interviews using AI-driven analysis, live feedback, and real-time audio visualizers.*

#### 💡 Key Features
- **AI Mock Interview Simulation** – Implements an interactive voice-to-voice simulation with dynamic, adaptive questioning.
- **Resume Analyzer & ATS Optimization** – Analyzes resumes against job descriptions to provide detailed ATS matching scores and action items.
- **Coding Interview Practice** – A live coding workspace with real-time feedback on algorithmic efficiency and complexity.
- **Pro-Interactive Multilingual Voice Mentor** – A global overlay workspace with native multi-language voice support (Hindi, Telugu, Tamil, Kannada, Malayalam, Marathi, Bengali, and Gujarati).
- **Performance Analytics Dashboard** – Comprehensive tracking of historical user progress, metric scores, and personalized learning pathways.

- **Databases:** MongoDB • Firebase
